💾 Backup dos Dados
Faça backup regularmente. Se o navegador for limpo, os dados são perdidos.
⚠️ Importante: Os dados ficam salvos no navegador (localStorage). Se limpar o histórico do navegador ou trocar de computador, os dados somem. Faça backup toda semana e guarde o arquivo em lugar seguro (Google Drive, WhatsApp, email).

📤 Exportar Backup
Gera um arquivo .json com todos os dados do sistema — clientes, pedidos, estoque, financeiro, produção e tudo mais.
Salve em um lugar seguro. Você pode importar de volta a qualquer momento.

📥 Importar Backup
Restaura todos os dados de um backup anterior.
⚠️ Atenção: A importação substitui todos os dados atuais. Faça um backup antes de importar.
